# Pythonik-net Task Tracker

## Overview
Pythonik-net developed this Task Manager application for TaskFlow Solutions to provide a dynamic and user-friendly way to manage tasks.

## Features
- Add, edit, and delete tasks
- Assign priority levels
- Mark tasks as important or completed
- Display task creation date
- Dynamic DOM updates
- JSON console logging of all task actions

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Open index.html in a browser
2. Enter a task and select options
3. Click "Add Task" to deploy

## Project Structure
- index.html → UI layout
- style.css → styling
- app.js → logic and functionality

## Notes
This project is inspired by Pythonik-net workflow systems but implemented as a client-side web application for instructional purposes.
